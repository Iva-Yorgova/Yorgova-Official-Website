SUMMER STORE is an Angular Web Application that combines e-commerce store for ordering favorite summer sweets and a blog, where users can read, write and comment interesting articles. The project use SNIPCART - HTML/JavaScript shopping cart platform, which is flexible, developer-friendly and easy to use. Application data is managed by integrating Firebase authentication service, storage service and hosting. Interaction with The Pexels RESTful JSON API on the photos page, enables users to search Pexels for any topic that they would like.
